FE+, Lection 2, task 1

Задача

Создать функцию runAsyncAll(list, async), которая принимает массив функций с асинхронными операциями и параметр async. Если async == true, то все функции запускаются параллельно и runAsyncAll возращает массив результатов (от всех асинхронных операций). Если async == false, то все функции запускаются синхронно по порядку, и runAsyncAll возращает массив результатов (от всех асинхронных операций).
